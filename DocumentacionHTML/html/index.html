Guía Rápida

Ver todas las clases Haz clic en la pestaña "Clases" en el menú superior.

Explorar archivos Haz clic en "Archivos" para ver todos los scripts.

Buscar algo específico Usa la barra de búsqueda en la parte superior.

Ejemplos de Código

Crear una instancia de HealthSystem 

HealthSystem health = gameObject.AddComponent<HealthSystem>();
health.maxHealth = 100f;
health.currentHealth = 100f;

Configurar movimiento del jugador 

PlayerMovement movement = GetComponent<PlayerMovement>();
movement.speed = 5f;
movement.jumpForce = 10f;
