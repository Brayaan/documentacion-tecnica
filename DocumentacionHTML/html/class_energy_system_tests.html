Detailed Description

TEST SUITE: EnergySystem.

PROPÓSITO: Validar el sistema de energía del jugador en combate.

Este sistema controla cómo se gana y consume energía a través de:

  • Ataques (puñetazo, patada)
  • Recibir daño
  • Bloquear

También valida límites:

  • No puede superar el máximo de energía
  • No puede bajar de 0

DEPENDENCIA IMPORTANTE: EnergySystem depende de UI (energyImage), pero en tests se deja null. Esto genera LogError controlado que se ignora con LogAssert.
