◆ UpdateEnergyUI()

void EnergySystem.UpdateEnergyUI ( )
inlineprivate

Actualiza el sprite de la barra de energía en la UI según currentEnergy. Calcula el índice proporcional en el spritesheet e invierte el orden para que el sprite 0 del array corresponda a energía llena.

CUÁNDO SE EJECUTA: Llamado internamente después de cualquier cambio en currentEnergy.

LÓGICA DE INVERSIÓN: index normal = 0 → energía vacía (sprite de barra vacía) index invertido = (Length-1) - index → invierte el mapeo Resultado: index 0 del array = barra llena
