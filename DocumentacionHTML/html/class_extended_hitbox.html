void ExtendedHitbox.OnTriggerEnter2D ( Collider2D other)
inlineprivate

OnTriggerEnter2D — Se ejecuta una vez al entrar en contacto con otro Collider2D. Registra el impacto al PRIMER frame de contacto (no continuo como OnTriggerStay2D).

CUÁNDO SE EJECUTA: Cuando el Collider2D de este hitbox toca otro collider. QUÉ CONTROLA: Validaciones de ataque activo, aplicación de daño y ganancia de energía.

FLUJO INTERNO:

  1. Verificar que las referencias críticas existen.
  2. Verificar que PlayerAttack está en estado de ataque activo.
  3. Verificar que el collider tocado tiene el tag "Enemy".
  4. Obtener EnemyHealthSystem y aplicar daño con el valor del Hitbox padre.
  5. Otorgar energía al atacante por conectar el golpe.
