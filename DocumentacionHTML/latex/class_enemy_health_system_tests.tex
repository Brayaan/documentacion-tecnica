TEST SUITE:\+ \doxylink{class_enemy_health_system}{Enemy\+Health\+System}. 

PROPÓ\+SITO:\+ Validar el sistema de vida de los enemigos, incluyendo:\+
\begin{DoxyItemize}
\item Aplicación de daño
\item Límites de vida (no menor a 0)
\item Knockback al recibir daño
\item Hit-\/stun (inmunidad temporal)
\item Muerte del enemigo cuando la vida llega a 0
\end{DoxyItemize}

USO:\+ Estas pruebas simulan enemigos en runtime y verifican su comportamiento sin necesidad de escena Unity.

DEPENDENCIAS:\+
\begin{DoxyItemize}
\item Rigidbody2D (para knockback físico)
\item Unity Physics2D (Fixed\+Update)
\item Destrucción de Game\+Objects (Destroy) 
\end{DoxyItemize}
