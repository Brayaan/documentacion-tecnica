Este conjunto de pruebas valida la lógica de ataque del jugador:\+
\begin{DoxyItemize}
\item Activación y desactivación de hitboxes
\item Control de cooldowns
\item Estado de ataque (is\+Attacking)
\end{DoxyItemize}

LIMITACIÓN:\+ Unity Input no se puede simular directamente en Play\+Mode, por lo que la lógica se prueba llamando métodos internos (Invoke, API pública y estados internos).
