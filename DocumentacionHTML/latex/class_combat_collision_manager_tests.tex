\doxysubsection{Descripción detallada}
TEST SUITE:\+ \doxylink{class_combat_collision_manager}{Combat\+Collision\+Manager}. 

PROPÓ\+SITO:\+ Verificar que el sistema de combate evita la colisión física entre enemigos cuando la configuración enemies\+Push\+Each\+Other está desactivada.

USO:\+ Este test simula dos enemigos en escena y valida que Unity Physics2D ignore la colisión entre ellos cuando el manager lo indica.

LIMITACIONES:\+
\begin{DoxyItemize}
\item Depende de Physics2D (motor de físicas de Unity).
\item No usa escena real; se construye todo en runtime.
\item Requiere Rigidbody2D + Collider2D para simular interacción. 
\end{DoxyItemize}
