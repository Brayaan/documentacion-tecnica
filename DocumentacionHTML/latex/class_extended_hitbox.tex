On\+Trigger\+Enter2D — Se ejecuta una vez al entrar en contacto con otro Collider2D. Registra el impacto al PRIMER frame de contacto (no continuo como On\+Trigger\+Stay2D). 

CUÁ\+NDO SE EJECUTA:\+ Cuando el Collider2D de este hitbox toca otro collider. QUÉ CONTROLA:\+ Validaciones de ataque activo, aplicación de daño y ganancia de energía.

FLUJO INTERNO:\+
\begin{DoxyEnumerate}
\item Verificar que las referencias críticas existen.
\item Verificar que \doxylink{class_player_attack}{Player\+Attack} está en estado de ataque activo.
\item Verificar que el collider tocado tiene el tag "{}\+Enemy"{}.
\item Obtener \doxylink{class_enemy_health_system}{Enemy\+Health\+System} y aplicar daño con el valor del \doxylink{class_hitbox}{Hitbox} padre.
\item Otorgar energía al atacante por conectar el golpe. 
\end{DoxyEnumerate}
