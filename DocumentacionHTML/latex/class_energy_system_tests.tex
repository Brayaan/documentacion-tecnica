\doxysubsection{Descripción detallada}
TEST SUITE:\+ \doxylink{class_energy_system}{Energy\+System}. 

PROPÓ\+SITO:\+ Validar el sistema de energía del jugador en combate.

Este sistema controla cómo se gana y consume energía a través de:\+
\begin{DoxyItemize}
\item Ataques (puñetazo, patada)
\item Recibir daño
\item Bloquear
\end{DoxyItemize}

También valida límites:\+
\begin{DoxyItemize}
\item No puede superar el máximo de energía
\item No puede bajar de 0
\end{DoxyItemize}

DEPENDENCIA IMPORTANTE:\+ \doxylink{class_energy_system}{Energy\+System} depende de UI (energy\+Image), pero en tests se deja null. Esto genera Log\+Error controlado que se ignora con Log\+Assert.
