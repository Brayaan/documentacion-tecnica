Detailed Description

TESTS: HealthSystem.

PROPÓSITO: Validar el sistema de vida del jugador:

  • Reducción de vida
  • Límite mínimo (0)
  • Knockback físico
  • Estado de hitstun
  • Restauración del control del jugador

USO: Ejecutar en Unity Test Runner (PlayMode). Cada test crea su propio jugador aislado.
