◆ Update()

void PlayerAttack.Update ( )
inlineprivate

Update() — Se ejecuta cada frame. Lee el input de teclado para puñetazo y patada. Verifica cooldown, flag de ataque activo y disponibilidad antes de atacar.

CUÁNDO SE EJECUTA: Cada frame durante la partida. QUÉ CONTROLA: Input de ataques, inicio de StartAttack(), activación de hitboxes, triggers de animación y cooldowns.

ORDEN DE VERIFICACIÓN (puñetazo):

  1. Input detectado (GetKeyDown).
  2. No hay ataque en curso (!isAttacking).
  3. Puñetazo disponible (punchReady). → StartAttack() + animación + hitbox activo + cooldown.
